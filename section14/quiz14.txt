Date Functions

Question 1: 

How do you determine the current date and time using the datetime object from the datetime module?

    from datetime import datetime

	Answer: datetime.now()

Question 2:

How do you create a datetime object for the exact date: 2018-04-01, 06:58:47?

    from datetime import datetime

	Answer: datetime(2018, 4, 1, 6, 58, 47)


Question 3:

How do you create a date display for the datetime object day created by day = datetime.now() in the following form?

Monday, 12th of January

    from datetime import datetime
     
    day = datetime.now()

Check here in remembrance: https://docs.python.org/3.6/library/time.html

	Answer: day.strftime("%A, %dth of %B")

Rationale: 
%A Locale’s full weekday name.
%d Day of the month as a decimal number [01,31].
%B Locale’s full month name.
